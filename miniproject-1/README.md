# Mandelbrot Set Visualization

This script generates and visualizes the **Mandelbrot Set**, one of the most famous fractals in mathematics.

## What is the Mandelbrot Set?

The Mandelbrot set is the set of complex numbers `c` for which the iterative function:

```
z_{n+1} = z_n^2 + c
```

does not diverge when starting from `z_0 = 0`. In other words, the sequence stays bounded (|z| ≤ 2) indefinitely.

## How the Code Works

### 1. **Setup Parameters**
```python
xmin, xmax = -2.0, 1.0    # Real axis range
ymin, ymax = -1.5, 1.5    # Imaginary axis range
width, height = 1024, 1024 # Image resolution (pixels)
max_iter = 100             # Maximum iterations to test
```

- **x/y ranges**: Define the viewing window in the complex plane (Re + Im)
- **width/height**: Higher values = more detail but slower computation
- **max_iter**: More iterations = better accuracy for determining membership

### 2. **Create Complex Grid**
```python
x = np.linspace(xmin, xmax, width)
y = np.linspace(ymin, ymax, height)
X, Y = np.meshgrid(x, y)
c = X + 1j * Y
```

Creates a 2D grid of complex numbers `c = Re + Im*j`, where each pixel represents one complex number to test.

### 3. **Iterate the Mandelbrot Formula**
```python
for n in range(max_iter):
    mask = np.abs(z) <= 2
    z[mask] = z[mask] ** 2 + c[mask]
    output[mask & (np.abs(z) > 2) & (output == 0)] = n
```

For each point:
- Start with `z = 0`
- Repeatedly compute `z = z^2 + c`
- If `|z| > 2`, the point escapes (not in the set)
- Record **how many iterations** before it escaped
- Points that never escape (after max_iter) are in the Mandelbrot set

### 4. **Visualization**
```python
plt.imshow(output, extent=[xmin, xmax, ymin, ymax], cmap='hot')
```

- **Black/dark regions**: Points in the Mandelbrot set (never escaped)
- **Colored regions**: Points outside the set (color = iteration count at escape)
- The intricate boundary creates the famous fractal pattern

## Interactive Controls

### Buttons
- **Zoom In (+10 iter)**: Zooms to the center by 2x and increases iterations by 10 for better detail
- **Zoom Out (-10 iter)**: Zooms out by 2x and decreases iterations by 10 (minimum 10)
- **Reset**: Returns to the original view

### Click to Zoom
**Left-click** anywhere on the fractal to zoom into that location (automatically increases iterations by 10)

### Why Adjust Iterations?
- **Zooming in** reveals finer details that require more iterations to determine if points escape
- **Zooming out** shows larger structures that need fewer iterations
- This automatic adjustment ensures optimal detail at each zoom level

## Mathematical Background

The Mandelbrot set was discovered by Benoit Mandelbrot in 1980 and is famous for its:
- **Self-similarity**: Zooming in reveals similar patterns at all scales
- **Infinite complexity**: The boundary has infinite detail
- **Simple definition**: Despite complexity, defined by a simple iterative formula

The set is connected to chaos theory, complex dynamics, and has applications in computer graphics, physics, and art.
